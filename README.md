# Softaculous WordPress SSO - WHMCS Addon Module

מודול WHMCS להתחברות בקליק לאתרי וורדפרס דרך Softaculous.

## תכונות

### איזור לקוח
- 🔐 **התחברות בקליק** - כניסה ישירה לדשבורד וורדפרס ללא צורך בסיסמה
- 📋 **רשימת אתרים** - הצגת כל אתרי הוורדפרס של הלקוח
- 🔌 **ניהול תוספים** - הפעלה, כיבוי, העלאה ומחיקה של תוספי וורדפרס
- 🎨 **ניהול תבניות** - הפעלת תבניות וורדפרס
- 🔄 **תמיכה ב-cPanel ו-DirectAdmin** - זיהוי אוטומטי של סוג השרת
- 🎨 **עיצוב מודרני** - ווידג'ט יפה שמשתלב בעמוד פרטי המוצר

### איזור ניהול (Admin)
- 👁️ **צפייה בהתקנות** - רשימת כל התקנות הוורדפרס של הלקוח
- 🔌 **ניהול תוספים מלא** - הפעלה, כיבוי, העלאה ומחיקה של תוספים
- 🎨 **ניהול תבניות** - הפעלת תבניות עבור הלקוח
- 🔐 **התחברות כמנהל** - כניסה לדשבורד וורדפרס של הלקוח

## התקנה

1. העלה את תיקיית `softaculous_sso` לנתיב:
   ```
   /path/to/whmcs/modules/addons/
   ```

2. היכנס לממשק הניהול של WHMCS:
   - Setup → Addon Modules
   - מצא את "Softaculous WordPress SSO"
   - לחץ על "Activate"
   - הגדר את הפורטים לפי הצורך

## הגדרות

| הגדרה | ברירת מחדל | תיאור |
|--------|-------------|--------|
| פורט cPanel | 2083 | פורט ברירת מחדל לחיבור ל-cPanel |
| פורט DirectAdmin | 2222 | פורט ברירת מחדל לחיבור ל-DirectAdmin |
| פורט מותאם אישית | - | אם מוגדר, ישתמש בפורט זה לכל השרתים |

## דרישות

- WHMCS 7.0 ומעלה
- Softaculous Premium מותקן על השרתים
- PHP 7.2 ומעלה
- cURL extension

## שימוש

### איזור לקוח
לאחר ההתקנה וההפעלה, הלקוחות יראו טבלה בעמוד פרטי המוצר שלהם עם:
- כתובת האתר
- גירסת וורדפרס
- כפתור "התחברות" לכניסה ישירה לדשבורד
- כפתור "תוספים" לניהול תוספי וורדפרס
- כפתור "תבניות" לניהול תבניות וורדפרס

### איזור ניהול
בעמוד סיכום לקוח (Client Summary) יופיעו כפתורים:
- **התקנות וורדפרס** - צפייה ברשימת ההתקנות והתחברות
- **ניהול תוספים** - הפעלה, כיבוי, העלאה ומחיקה של תוספים
- **ניהול תבניות** - הפעלת תבניות
